# Trabajo 1: CSS

## Descripción
Este proyecto consiste en una página web interactiva creada para explicar y demostrar algunos de los principales conceptos de **CSS** de forma visual y entretenida.  

La idea es que los usuarios puedan **ver ejemplos interactivos y ver cómo funcionan las distintas características de CSS vistas en clases**.

---

La página incluye demostraciones y explicaciones de los siguientes elementos de CSS moderno:

- **Media Queries**
- **Container Queries**
- **Flexbox**
- **CSS Grid**
- **Variables CSS (Custom Properties)**
- **Funciones `calc()`**
- **Anidación de reglas (CSS Nesting)**
- **Transformaciones (`transform`)**
- **Transiciones (`transition`)**
- **Animaciones (`@keyframes`)**

Cada sección tiene:
- una **demostración interactiva**
- una **explicación del concepto**
- **ejemplos de código**

Además, se incluye un **quiz** al final para practicar los conceptos vistos.
